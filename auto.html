Automation

Here at Los Santos Life Australia, we have a lot of different jobs to offer. Some of which, can be AFK'able in a sense.
For instance, Fishing, Lumbering, Fueling and Tailoring typically have 3-4 steps invloved, from collection, to finishing touches to selling.
These can be completed in an AFK'able method where a macro can be activated and can then complete the steps with minimal human requirements.

Principles of macros

Macros work by completing a series of keypresses with timings between them in order to be used to essenetially complete a task automatically or easier.
Macros will not work with recording mouse movements (only clicks). Therefore it's better to find out how to complete the task using keyboard only. Scripts would have to be used in order to complete tasks where mouse movements or entering specific fields are required (which is a lot more likely to get you banned).

Unfortunately, without scripts, we're unable to leave the GTA window, so you can't use your computer while this method is being utilised. If set up correctly, the only thing you'll need to return to complete, is relocating one each step has been complete - to go to the next location. Each step will likely require a different macro.

Setting up a macro

First of all, make sure you have a device that can utilise macros, if not, you may require installing AutoHotKey (AHK) and assign the macro to a series of keypresses.
Most hardware (mouse/keyboard) software are quite similar, however they do range in their functions. Software like Razer Synapse, Steelseries Engine and Corsair iCUE etc. all allow you to create macros and some even have the functionality of recording them on the fly.

Run through the first task you want to automate (collection) and take notes on what keys are pressed along with the (rough) time between them. Don't worry if you miss something, it can be added later.

Step Action
1Record the keys inside your software
2Edit timings, assign the macro to a key and test in-game
3Make notes on how the macro is responding, you should be able to relate the keypresses to your notes
4Final touches to macro
5Set up recurring macro

Repeat this process for all other tasks (that can be automated) and assign to keys.
